Environment Setup

Get your laptop ready for the hands-on labs

Recommended – Quick Start

cd lecture_slides/3.day2_before_lunch/notebooks
bash setup_env.sh
conda activate genomic-llm-finetuning

1. Requirements

  • Python 3.11+
  • Conda or Mamba
  • 8–16 GB RAM
  • ~4–6 GB disk space
  • GPU strongly recommended (Day 2)

2. Manual install (if needed)

conda create -n genomic-llm-finetuning python=3.11 -y
conda activate genomic-llm-finetuning

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
pip install transformers==4.45.* peft==0.12.* bitsandbytes accelerate
pip install jupyter pandas numpy scikit-learn matplotlib seaborn umap-learn

GPU access tip

Day 2 fine-tuning is much faster with a GPU.
No local GPU? → Use Google Colab (free T4/P100) or Kaggle notebook.
